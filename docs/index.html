ETHEREAL

Fast, simple, easy library to work with colors: single, palettes, gradients and more. Written in Typescript.

See Demo View on Github

Ethereal Color is a library without dependencies to generate colors, create palettes and gradients. Everything here works with simple functions and is 100% free and open source, in addition to supporting intellisense via Typescript.

For documentation and a full list of available features, visit the Github page.
